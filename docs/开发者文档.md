# 舌镜辨证能力引擎 · 开发者文档

> **文档定位**：本文档面向开发者和AI Agent，提供舌镜辨证API接入指南。

## 概述

舌镜是辨证能力引擎，提供舌象识别与中医辨证分析API。通过上传舌象图片，返回结构化的舌诊数据，可对接本地辨证引擎实现完整的中医智能诊断。

### 接入方式

| 方式 | 适用场景 | 文档位置 |

| **MCP 接入** | Claude Desktop、Cursor等MCP客户端 | 见下方 |
| **扣子 Bot** | Coze平台对话式舌诊 | 见下方 |
| **HTTP API** | 自有系统对接 | 见下方 |

## 已开放接口

### 接口1：舌象上传分析

**端点**：`POST /api/tongue-ai/tongue`

上传舌象图片，创建舌诊分析任务。

**请求参数**：

```json
{
  "image": "data:image/jpeg;base64,..."  // Base64编码的图片
}
```

**成功响应**：

```json
{
  "success": true,
  "chat_id": "xxx",
  "conversation_id": "xxx",
  "status": "in_progress"
}
```

---

### 接口2：查询分析结果

**端点**：`GET /api/tongue-ai/result?chat_id=xxx&conversation_id=xxx`

轮询查询舌诊分析结果。

**处理中响应**：

```json
{
  "success": true,
  "status": "in_progress"
}
```

**完成响应**：

```json
{
  "success": true,
  "status": "completed",
  "data": {
    "tongueDetected": true,
    "tongue_color": "淡红",
    "tongue_shape": "淡胖",
    "tongue_coating": "薄白",
    "region_features": "...",
    "shape_distribution": "..."
  }
}
```

---


## MCP 接入

适用于 Claude Desktop 等 MCP 客户端。

**Server Address**：`https://mcp.zheng-he.top/mcp`

**可用工具**：

| 工具名 | 功能 |
|--------|------|
| `tongue_analyze` | 舌诊辨证分析 |
| `tongue_interrogate` | 问诊交互 |
| `acupoint_query` | 穴位查询 |

### Claude Desktop 配置

在 `~/Library/Application Support/Claude/claude_desktop_config.json` 中添加：

```json
{
  "mcpServers": {
    "tongue-mirror": {
      "url": "https://mcp.zheng-he.top/mcp"
    }
  }
}
```
